# Portfolio Peu

Portfolio interativo desenvolvido para apresentar Peu como estrategista criativo, publicitario, criador de conteudo e especialista em cultura digital, memes, tendencias e grandes audiencias.

## Acesso

O projeto publicado pode ser acessado em:

[https://wendell-ramos.github.io/PortfolioPeu/](https://wendell-ramos.github.io/PortfolioPeu/)

## Sobre o projeto

A proposta do site e transformar a trajetoria e o repertorio criativo do Peu em uma experiencia visual mais artistica, interativa e alinhada com o universo de cultura pop, internet brasileira, campanhas, comunidades e conteudo viral.

O portfolio destaca:

- trajetoria profissional e autoridade criativa;
- crescimento de comunidades e perfis de grande audiencia;
- cases como Alfinetei, Sou Eu na Vida, Banca Digital, BBB, BYD, eventos e Mynd & Billboard;
- hub criativo, leitura de tendencias e oportunidades de marca;
- redes sociais e contato profissional.

## Tecnologias

- HTML5
- CSS3
- JavaScript
- Canvas para efeitos visuais de fundo
- Layout responsivo para desktop, tablet e mobile

## Como rodar localmente

Como o projeto e estatico, basta servir a pasta com um servidor local:

```bash
python -m http.server 4173
```

Depois acesse:

[http://127.0.0.1:4173](http://127.0.0.1:4173)
